어떻게 나오니?
element는 여는 Tag와 닫는 Tag로 이루어진다. 속성(attribute)는 여는 Tag 옆에 입력하는 엘리먼트의 설정이다.
속성에서 많이 사용하는 것들
  1. id : 특정한 엘리먼트를 나타내는 이름, 하나의 엘리먼트에 하나의 ID를 설정할 수 있다. 중복될 수 없다.
  2. class : 엘리먼트의 클래스 이름을 정하여 묶음으로 설정할 수 있다. 중복될 수 있다.
  3. src : 파일을 가져올 때 설정한다. 보통 img, video, audio 엘리먼트에서 설정한다. source
  4. alt : 파일을 가져오기 못했을 때 출력하는 텍스트 설정
  5. charset : 데이터의 포멧을 설정, 유니코드 등
  6. type : 사용자가 입력한 데이터의 형식을 설정 text number file
  7. for : label 엘리먼트에서 사용되며 input의 id 속성을 찾아 연결한다. 라벨을 클릭 시 for에 설정된 값과 같은 input의 id를 찾아 입력할 수 있게 커서를 옮겨준다.
  8. href : a 엘리먼트, 하이퍼링크의 주소 또는 파일 위치를 입력한다.
  9. style : id, class 등과 마찬가지로 모든 엘리먼트에 설정 가능한 속성. 엘리먼트가 어떻게 출력될 것인지를 설정

블락 넣은 스타일 어떻게 나올까요?
엘리먼트는 기본적으로 열었으면 닫아야한다. 한줄로 된, 자식 엘리먼트가 없는 엘리먼트의 경우 바로 닫을 수 있다.(ex : input) 끝에 /가 들어감
cd 는 폴더를 이동하는 명령어. ".."은 상위 폴더로 이동한다.
windows에서는 dir 명령어를 사용해서 폴더 내 파일과 폴더 목록을 확인할 수 있다. Mac에서는 ls 명령어를 사용해서 dir과 같이 폴더 내 파일과 폴더를 확인할 수 있다.
img, video, audio, a 엘리먼트에서 사용하는 경로(src, href)에 대해서 ./는 현재 파일이 위치한 폴더를 기준으로 한다. ../는 현재 파일이 위치한 폴더의 상위 폴더로 이동해서 파일 또는 폴더를 찾게 된다.
